# docker-fluentd-examples
dockerを使ってfluentd(v1.2)を試した。

+ httpリクエストをファイルに書き出す  
  [http2file](./http2file)
+ nginxのアクセスログをファイルに書き出す  
  [nginx2file](./nginx2file)
+ nginxのアクセスログをMySQLに入れる  
  [nginx2mysql](./nginx2mysql)
+ nginxのアクセスログをBigQueryに入れる  
  [nginx2bq](./nginx2bq)

## 必要なもの
+ docker
+ docker-compose

各ディレクトリ内のREADMEを参照
